微软将自由软件基金会列为“赌博网站”

Jun 26th, 2012 | 评论

自由软件基金会（FSF）执行官Jonh Sullivan近日公开抱怨称，微软网站声望数据库（reputation database）将FSF的网站donate.fsf.org列为“赌博网站”。有用户反馈由于过滤器的拦截无法在自己的工作环境下访问。

经检验，发现微软将其标记为“赌博”、“科技信息”和“共享/免费软件”网站。Sullivan称自己已将该错误分类反馈给微软并得到回复，但至今该错误仍未被修正。

如果用户因此不能访问，可以尝试直接点击以下链接：my.fsf.org/donate。

消息来源：http://www.h-online.com/open/news/item/Microsoft-filter-blocks-Free-Software-Foundation-donation-site-1624934.html

MuseScore：免费的作曲与乐谱软件

Apr 28th, 2012 | 评论

真没想到在这方面也会有如此强大好用的跨平台软件。这果然验证了“好东西必然开放”的道理。

官网的介绍：

MuseScore 是一个免费的可运行在多种平台上的 所见即所得 的音乐制谱软件，它提供一个专业制谱软件如Sibelius，Finale和Capella的高性价比的替代方案.
你可以打印出排版的很漂亮的乐谱, 或保存为 PDF 和 MIDI 文件.

一些亮点:

  • WYSIWYG（所见即所得）, 在"虚拟乐谱纸"上输入音符
  • 不限制谱表个数
  • 多声部：每个谱表最多支持四个声部
  • 简便快速的使用鼠标、键盘或MIDI键盘输入音符
  • 使用内置的音序器或软件合成器 FluidSynth 发声
  • 可读取或保存为MusicXML 或标准 MIDI 文件
  • 能运行在 Windows，Mac 和 Linux 系统上
  • 已翻译为 47 种语言版本
  • 遵循 GNU GPL 许可
并且看上去支持还不错，大家可以试一试。

我心飞去

Mar 24th, 2012 | 评论

终于有一天，我发现我其实并不属于这里。这里的生活太过于泥泞、肮脏，以至于我怀疑我当初为什么会做出这样的选择，来到这样一个鬼地方，或许，本来就不是我想来这里的。地狱也不过如此，我想。

我此前并不知道，除了我以外，还有那么多人在这条路上受着苦。密布乌云的天空下着连绵不绝的雨，人人都打着伞，人人都被淋湿。人人都低着头，用属于自己的伞遮住自己的脸，遮住前面的路。高高矮矮胖胖瘦瘦快快慢慢一样的人，大大小小新新旧旧长长短短一样的伞，一样的行走方向，一样的路。人人都以为只有自己受着最真切的苦难，摆出乞求施舍的样子。而我，放下伞，偶然停下来，抬头望天，嗔视天降的雨滴，才知道一切路人们也都正受着永恒的苦难。不论是奉谁的意愿，这都是为在这条路上行走的人们准备好了的。而我们一定不属于这里。

中国古典戏剧小报制作有感

Feb 10th, 2012 | 评论

早就领教过家庭伦理电视剧的威力，对于婚姻、爱情的理解可谓是越来越肤浅，以至于对于这类题材的作品失去兴趣，处于麻木的状态。

也许是这样，这就是所谓现代都市化生活的副作用吧。我们有一大堆情景喜剧可看，已经很少有人会特意去看这样的古典戏剧了，更何况还是个悲剧。

难得有这样一个机会，有去看这样的戏剧的动机。事实上也成为从繁忙之中脱身的一个借口。不管怎么说，我可以离开现实世界一会儿，哪怕是几个小时。

欣赏完——或者说参与了——这部戏后，我才发现，原来这类悲剧都可以称得上是治愈系的了。世间人与人之间的关系原来还可以如此单纯！纯美？清新？无瑕？完美？这是一个桃花源。你该如何形容发现一个桃花源时的喜悦呢？

每个人心中都有一个桃花源。不过，在奔跑的路上，有的已经被遗忘。只要再唤起对它的记忆，一样是美好的。尘世中美好的事，不过是心中留有一处桃花源。

RQNOJ 137 找试场

Feb 10th, 2012 | 评论 

#include <stdio.h>
#include <ctype.h>
#include <stdlib.h>
int x=0,y=0;
//dir方向：1是右，2是左，3是上，4是下。
int dir=3;
//未移动标记。
int flag=0;
void setdir(char a)
{
	if(a=='l')
	{
		switch(dir)
		{
			case 1:
				dir=3;
				break;
			case 2:
				dir=4;
				break;
			case 3:
				dir=2;
				break;
			case 4:
				dir=1;
				break;
		}
	}
	if(a=='r')
	{
		switch(dir)
		{
			case 1:
				dir=4;
				break;
			case 2:
				dir=3;
				break;
			case 3:
				dir=1;
				break;
			case 4:
				dir=2;
				break;
		}
	}
	return;
}
void go(int dis)
{
	if(dir==1) x+=dis;
	if(dir==2) x-=dis;
	if(dir==3) y+=dis;
	if(dir==4) y-=dis;
	flag=1;
	printf("(%d,%d)\n",x,y);
	return;
}
int main()
{
	int n;
	scanf("%d",&n);
	int i;
	for(i=1;i<=n;i++)
	{
		char ins[20];
		scanf("%s",ins);
		if(isdigit(ins[0]))
			go(atoi(ins));
		if(ins[0]=='r')
			setdir('r');
		if(ins[0]=='l')
			setdir('l');
	}
	if(flag==0) printf("(0,0)\n");
	return 0;
}

#include <stdio.h>
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;
int main()
{
	string s;
	char in[20001];
	scanf("%s",in);
	s=in;
	reverse(s.begin(),s.end());
	string::iterator it=s.begin();
//此处需要注意，输入100须输出1。
	while(*it=='0')
		s.erase(it);
	cout<<s<<endl;
	long int sum=0;
	for(it=s.begin();it!=s.end();it++)
		sum+=(*it-'0');
	cout<<sum;
	return 0;
}
